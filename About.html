Levels

Complete the Tutorial and Levels 1-12 to reveal later levels.
  • Levels 1-12 build the toolbar and serve as an introduction to GeoGebra
  • Levels 13-18 pose typical construction problems for school mathematics
  • Levels 19-25 provide more advanced constructions, some of which are quite challenging
  • Levels 26-28 illustrate triangle congruence criteria
  • Levels 29-32 develop basic rigid motions and dilation
  • Level 33 recreates Euclid's Proposition 2, which "builds" a (non-collapsing) compass

Controls

  • Tools on the toolbar
  • Undo and redo arrows, upper right

Medals

If you complete a level, you receive a silver medal. The number inside the medal is the number of moves you used to complete the level. If you complete a level in a minimal number of moves, you receive a golden medal.

From Level 2 and onwards, you can get two different medals for a level: one for using all available tools, and another for using only the primitive tools available on Level 1.

About

Kasper Peulen's brilliant game has been borrowed and rehosted here at http://euclid.findell.org.  We have squashed bugs, edited the text and sketches, improved the GeoGebra functionality, and added new levels! Please report bugs and send suggestions to Brad Findell.
